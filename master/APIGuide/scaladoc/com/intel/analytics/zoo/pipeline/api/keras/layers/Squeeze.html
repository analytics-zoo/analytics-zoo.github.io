Squeeze 

Companion object Squeeze

class Squeeze[T] extends KerasLayer[Tensor[T], Tensor[T], T] with Net

Delete the singleton dimension(s). The batch dimension needs to be unchanged. For example, if input has size (2, 1, 3, 4, 1): Squeeze(dim = 1) will give output size (2, 3, 4, 1) Squeeze(dims = null) will give output size (2, 3, 4)

When you use this layer as the first layer of a model, you need to provide the argument inputShape (a Single Shape, does not include the batch dimension).

Remark: This layer is from Torch and wrapped in Keras style.

T

Numeric type of parameter(e.g. weight, bias). Only support float/double now.
