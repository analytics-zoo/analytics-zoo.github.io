AucScore 

class AucScore extends ValidationResult

Represent an accuracy result. It's equal to the probability that a classifier will rank a randomly chosen positive instance higher than a randomly chosen negative one. Refer: https://en.wikipedia.org/wiki/Receiver_operating_characteristic
