class SequenceShaper extends TextTransformer

Shape the sequence of indices to a fixed length. If the original sequence is longer than the target length, it will be truncated from the beginning or the end. If the original sequence is shorter than the target length, it will be padded to the end. Need to word2idx first. Input key: TextFeature.indexedTokens Output key: TextFeature.indexedTokens The original indices sequence will be replaced by the shaped sequence.

Instance Constructors

  1. new SequenceShaper(len: Int, truncMode: TruncMode = TruncMode.pre, padElement: Int = 0)

    len

    Positive integer. The target length.

    truncMode

    Truncation mode. Either TruncMode.pre or TruncMode.post. If TruncMode.pre, the sequence will be truncated from the beginning. If TruncMode.post, the sequence will be truncated from the end. Default is TruncMode.pre.

    padElement

    Integer. The index element to be padded to the sequence if the original length is smaller than the target length. Default is 0 with the convention that we reserve index 0 for unknown words.
